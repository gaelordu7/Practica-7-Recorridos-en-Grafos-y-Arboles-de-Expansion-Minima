# Práctica 7: Recorridos en Grafos y Árboles de Expansión Mínima

Este proyecto implementa algoritmos fundamentales de la teoría de grafos, incluyendo exploraciones sistemáticas y optimización de redes mediante Árboles de Expansión Mínima (MST).

## Estructura del Proyecto
- `grafo.py`: Implementación de la clase `Grafo` (no dirigido, no ponderado) mediante listas de adyacencia.
- `recorridos.py`: Algoritmos BFS, DFS y detección de ciclos.
- `mst.py`: Algoritmos de Prim (con min-heap) y Kruskal (con Union-Find) para grafos ponderados.
- `experimentos.py`: Script de comparación de rendimiento entre Prim y Kruskal.

## Requisitos
- Python 3.8+
